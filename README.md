# 길수매매법 코인 검색기 MVP

이 프로젝트는 업로드된 PDF의 핵심 원칙을 바탕으로 만든 **코인 추천 검색기**의 첫 번째 실행 가능한 버전입니다.

## 반영한 핵심 원칙
- 1시간봉 중심으로 해석한다.
- 1시간 이하에서는 일반 다이버전스보다 **다이버전스 연계(최소 3개 꼭지점)**를 우선한다.
- 애매한 경우 30분봉으로 한 단계 내려가 재확인한다.
- 4시간봉은 일반 다이버전스로 보조 확인한다.
- 피보나치 0.618 ~ 0.786 구간을 핵심 되돌림 구간으로 본다.
- 피보나치 1 이탈은 무효로 본다.
- RSI 중간 구간보다 극단 구간(과매도/과매수)에 가까운 꼭지점을 더 높게 평가한다.

## 현재 구조
- `GET /health` : 헬스체크
- `GET /scan/main` : 메인 기준 추천 검색
- `GET /scan/sub` : 서브 기준 추천 검색
- `GET /scan/symbol/{symbol}` : 특정 심볼 상세 판정

## 메인 / 서브 차이
### 메인
- 1시간봉에서 상승/하락 다이버전스 연계 우선
- 30분 확인 가산
- 4시간 일반 다이버전스 가산
- Fib 0.618~0.786 근접 필수에 가깝게 강하게 적용
- 거래량 증가 / 시총대신 대체 유동성 필터 / 최근 급등 제외 / 상단 저항 거리 체크

### 서브
- 일반 다이버전스도 허용
- Fib 근접 범위 완화
- 거래량/저항/과열 필터 완화
- 감시 후보군 성격

## 빠른 실행
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## 로컬 테스트
```bash
curl 'http://127.0.0.1:8000/scan/main'
curl 'http://127.0.0.1:8000/scan/sub'
curl 'http://127.0.0.1:8000/scan/symbol/BTCUSDT?mode=main'
```

## Render 배포
- GitHub에 업로드
- Render에서 Web Service 생성
- Python 환경으로 배포
- Start command: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`

## 한계
이 버전은 **MVP**다.
즉, PDF에서 말하는 “이쁜 그림”을 완벽 재현하지는 못한다. 그 부분은 결국 수치화 문제다.
현재는 아래 방식으로 최대한 기계화했다.
- 스윙 고저점 탐색
- RSI 기반 일반 다이버전스 및 연계 탐지
- Fib 구간 판정
- 거래량/저항/과열 점수화

실전 수준으로 끌어올리려면 다음이 추가되어야 한다.
1. 거래소 universe 확장 및 시총 데이터 연동
2. 백테스트
3. 신호 저장 DB
4. 스케줄러 및 알림 연동
5. 파라미터 튜닝
